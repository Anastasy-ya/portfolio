# frontandbeyond.com

## Планы по доработке: 

- Добавление звука, бинауральные ритмы, каждый новый присокдиняется на событие отрисовки шейдера, со временем угасает
- Блог (отдельная страница)
- Портфолио (отдельная страница)
- Форма обратной связи (готова, ждет бэкенда для отправки сообщений)
- Оптимизация
- Игровая механика. Пользователь добавляет новый элемент в матрицу и игра разрушается
- AR режим (A-frame). Меняем расположение матрицы с закругленного на плоское, показываем при наведении на плоскость, новые элементы не замещают существующие, а добавляются поверх предыдущего слоя. Выглядит как вертикальная фрактальная башня с бесконечным циклом создания. Старые слои скрываются из области видимости и уничтожаются
